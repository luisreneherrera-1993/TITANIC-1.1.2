El ciclo de vida CRISP-ML(Q)

Cada etapa está documentada dentro del cuaderno titanic_crisp_ml.ipynb, desde el planteamiento del problema hasta el monitoreo posterior al despliegue.

01

Negocio

Definir el problema de supervivencia y tarifa, y los criterios de éxito.

02

Datos

EDA: distribución de clases, valores faltantes, correlaciones.

03

Preparación

Imputación, codificación, escalado e ingeniería de variables.

04

Modelado

Regresión Logística y Regresión Lineal con pipelines de sklearn.

05

Evaluación

Accuracy, ROC-AUC, F1 · MAE, RMSE, R².

06

Despliegue

App en Streamlit + Landing Page + repositorio en GitHub.

07

Monitoreo

Reentrenamiento y trazabilidad de versiones del modelo.

Aprendizaje supervisado

Dos modelos, dos preguntas

Ambos se implementan como Pipeline de scikit-learn con preprocesamiento incorporado, listos para servir predicciones en tiempo real desde Streamlit.

Clasificación

Regresión Logística

Predice si un pasajero sobrevive (0/1) a partir de clase, sexo, edad, tamaño familiar y tarifa. Usa codificación one-hot, imputación por mediana/moda y escalado estándar.

Cómo ejecutarlo

De tu terminal a producción

cd titanic-ml-project

pip install -r requirements.txt

# 1. Entrenar / explorar el modelo
jupyter notebook notebooks/titanic_crisp_ml.ipynb

# 2. Levantar el dashboard
streamlit run app/app.py
